Example

Been struggling with how move_and_collide() works. Do I have to multiply it by delta? no Here's what the documentation says: 

        This method takes one required parameter: a Vector2 indicating the body's relative movement. 
        Typically, this is your velocity vector multiplied by the frame timestep (delta). If the engine detects a collision 
        anywhere along this vector, the body will immediately stop moving. If this happens, the method will return a 
        KinematicCollision2D object.
Taken from Godot Official Documentation

delta here is the time difference between the current frame and the last.
